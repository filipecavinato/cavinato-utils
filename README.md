# 🛠️ cavinato-utils

Biblioteca utilitária pessoal com funções reutilizáveis para agilizar o desenvolvimento de projetos Python (CLI e GUI).

> **Status:** Em desenvolvimento | Versão 0.1.0

## 🚀 Funcionalidades

* **Menus Variados:** Menus elegantes e pré definidos para agilizar a criação de sistemas e/ou scripts.
* **Geradores:** Automatização para criação de senhas seguras e futuramente outros scripts.

## 📥 Instalação

Como esta biblioteca é privada e hospedada no GitHub, você pode instalá-la em seus projetos usando:

```
pip install git+https://github.com/filipecavinato/cavinato-utils.git@main
```
## 💡 Como usar

Exemplo rápido de importação e uso no seu código:

- Importar Modulo Completo:
```
from utilidades import nome_do_modulo
```
- Importar apenas uma Função:
```
from utilidades.nome_do_modulo import nome_da_função
```
- Exemplo de chamada de função
```
resultado = nome_do_modulo.nome_função("argumento")
print(resultado)
```
